# Решение кейс-задачи № 4

Данный алгоритм находит максимальную мощь стаи драконов, используя математически оптимальную стратегию разбиения общего количества голов.

Основная логика:
Оптимальное разбиение на тройки - для максимизации произведения наиболее выгодно разбивать головы на группы по 3
Корректировка остатка - если после деления на 3 остаётся 1 голова, выгоднее заменить одну тройку на две двойки (так как 2×2=4 > 3×1=3)
Вычисление результата - итоговое произведение вычисляется как 3^k × остаток

Ключевые особенности:
Использует математически доказанный оптимальный подход
Время работы: O(1) - только арифметические операции
Корректно обрабатывает все случаи через анализ остатка от деления
Не требует хранения дополнительных массивов данных

Примеры работы:
M=6: 3+3 → 3×3=9
M=7: 3+2+2 → 3×2×2=12
M=8: 3+3+2 → 3×3×2=18
M=10: 3+3+2+2 → 3×3×2×2=36
